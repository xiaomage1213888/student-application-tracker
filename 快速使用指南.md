## 🎯 快速体验步骤

### 1️⃣ 访问系统
打开浏览器，访问：**http://localhost:5173**

### 2️⃣ 老师端功能体验

1. **登录老师账号**
   - 用户名：`admin`
   - 密码：`admin123`

2. **查看数据看板**
   - 查看整体统计数据
   - 查看各班级对比
   - 查看进展分布图表

3. **学生管理**
   - 查看已导入的学生名单
   - 可以导入新的 Excel/CSV 文件

4. **投递记录管理**
   - 查看所有学生的投递记录
   - 支持筛选和导出 Excel

### 3️⃣ 学生端功能体验

1. **切换账号登录**
   - 退出当前账号
   - 使用学生账号登录（如 `student1` / `123456`）

2. **查看个人看板**
   - 查看个人投递统计
   - 查看进展分布
   - 查看投递趋势

3. **添加投递记录**
   - 点击"投递记录"菜单
   - 点击"新增投递"按钮
   - 填写公司信息、岗位等
   - 可以设置重视度、备注等

4. **管理投递记录**
   - 编辑已有记录
   - 更新面试进展
   - 删除无效记录

## 🔧 老师导入名单功能

### 当前状态
系统已经预置了 5 个学生账号，可以直接使用。

### 如需导入新名单
1. 使用老师账号登录
2. 进入"学生管理"页面
3. 点击"导入名单"
4. 上传 Excel/CSV 文件

**文件格式**：
```csv
name,className
张三，2024 级计算机 1 班
李四，2024 级软件工程 2 班
```

或 Excel 文件包含两列：`name` 和 `className`

## 📝 注意事项

1. **端口占用**：确保 3000 和 5173 端口未被占用
2. **数据持久化**：测试数据保存在 `backend/data/database.sqlite`
3. **账号安全**：测试环境请勿使用真实密码
4. **浏览器兼容**：推荐使用 Chrome、Edge 等现代浏览器
